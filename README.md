This repo exists as a generic verification environment
Tools used: Questasim, CMake, python

Prerequisites: Need to install Questasim from the official altera website. They offer a free Questasim Starter Edition with a free license
You must also install python

Currently this setup is only used to verify an asynchronous fifo and all of the coding is done in a single tb.sv file
However im hoping to improve this folder structure by making this env very easy to manage, add new RTL, new yamls , new testcases
Might even try adding this into future repos just to see if i can access some RTL from another repo, a useful thing to learn if im to verify bigger RTL in the future

If you want to contribute just drop me a message. Please do all your work in a branch and send it for merge i will review. :)
